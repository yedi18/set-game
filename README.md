# SET — משחק הקלפים

משחק SET מלא בדפדפן, קובץ HTML אחד — ללא תלויות חיצוניות.

## הרצה

פתח את `index.html` בכל דפדפן מודרני. אין שרת, אין התקנות.

## על המשחק

SET הוא משחק קלפים שבו כל קלף מוגדר על ידי **4 מאפיינים**:

| מאפיין | ערכים |
|--------|-------|
| מספר   | 1, 2, 3 |
| צבע    | אדום, ירוק, סגול |
| צורה   | יהלום, גל, עיגול |
| מילוי  | מלא, פסים, ריק |

**סט תקין** = 3 קלפים שבכל מאפיין, הערכים כולם זהים או כולם שונים.

## פיצ'רים

### מסך בית
- הצגת שיאים אישיים (ניקוד, זמן, רצף, משחקים)
- הסבר חוקים + דוגמת סט תקין
- הגדרות: סאונד, מצב עיוור-צבעים, קיצורי מקלדת
- כפתור "המשך משחק" אם משחק פעיל

### גיימפליי
- לוח של 12 קלפים (עד 21 כשמוסיפים)
- לחיצה על 3 קלפים → בדיקה אוטומטית עם אנימציה
- לוח חוזר ל-12 אחרי מציאת סט (גם כשהיו 15)
- עצירה + חזרה למסך הבית בכל עת

### ניקוד
- **+3 נקודות** לסט ללא רמזים
- **מכפיל רצף**: ×2 אחרי 3 סטים רצופים, ×3 אחרי 5
- **ניכוי לפי רמז**: −1 / −2 / −3 / −5

### מערכת רמזים (4 שלבים)
1. "יש סט בלוח?"
2. הדגשת קלף אחד מהסט
3. הדגשת שני קלפים
4. הצגת הסט המלא + תיאור מילולי

### שיאים אישיים (localStorage)
- ניקוד גבוה
- זמן מהיר
- רצף מקסימלי
- מספר משחקים + סטים שנמצאו סה"כ

### נגישות
- **מצב עיוור-צבעים**: מוסיף סמלים (◆ ≈ ●) מתחת לצורות
- **סאונד**: משוב קולי ב-Web Audio API (ניתן לכבות)

### קיצורי מקלדת

| מקש | פעולה |
|-----|--------|
| `1`–`9` | בחירת קלפים 1–9 |
| `Q`–`Y` | בחירת קלפים 10–15 |
| `N` | משחק חדש |
| `H` | רמז הבא |
| `D` | +3 קלפים |
| `Esc` | עצור / חזור לבית |

## טכנולוגיות

- HTML5 + CSS3 + JavaScript Vanilla
- SVG inline לציור קלפים
- Web Audio API לסאונד
- localStorage לשמירת שיאים והגדרות
